About Marshall is a stand alone application for SwimClubMeet. Use it to rally your swimmers for events.  Change the entrant's qualification status. Disqualify or scratch swimmers. Quick entry of club members into empty lanes - should you have (very) late arrivals.   Use Marshall in colaboration with Timekeeper to create a 'paperless' swimming session.   Written in Embarcadero RAD Studio (Pascal) using FMX it's ready for Multi-Platform delivery. Currently only compiled for Windows 64bit. It's form factor is designed for Microsofts TouchPad Pro.
